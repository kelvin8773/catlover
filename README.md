# robofriends
Tutorial for udemy course - React + Redux

Base on the tutorial of [Andrei Neagoie](https://github.com/aneagoie/robofriends-redux), I make a my style of robofriends.

reference link:
https://aneagoie.github.io/robofriends-redux/


To run the project:

1. Clone this repo
2. Run `npm install`
3. Run `npm start`
